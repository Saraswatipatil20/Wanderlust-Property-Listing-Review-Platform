# Wanderlust-Property-Listing-Review-Platform

Wanderlust is a full-stack property listing and review platform inspired by Airbnb, designed to provide users with a seamless experience for discovering, listing, and reviewing properties. It is built with Node.js, Express, MongoDB, and EJS, following the MVC architecture for maintainable and scalable code.



Key Features


Property Listings – Users can create, view, update, and delete property listings.

Review System – Users can post reviews and ratings for properties.

Image Upload – Upload multiple images for each listing with proper validation.

Error Handling & Middleware – Robust error handling with Express middleware.

RESTful API – Follows best practices for building scalable APIs.

Responsive UI – Designed with Bootstrap for a smooth user experience across devices.
